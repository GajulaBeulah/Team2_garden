Each product page (fruits, veg, pots) has buttons that need to be updated from:

OLD (plain button):

NEW (with product details):
<button onclick="addToCart('ProductName', Price, 'ImageURL', 1)">Add to Cart</button>

🔗 Available JavaScript Functions

// Add product to cart
addToCart(productName, price, imageURL, quantity)

// Remove product
removeFromCart(index)

// Increase/decrease quantity
updateQuantity(index, change)  // change: 1 (increase), -1 (decrease)

// Clear entire cart
clearCart()

// Proceed to checkout
checkout()

// Get cart data
getCartData()

// Display cart items
displayCartItems()

// Load cart from storage
loadCart()

💾 Data Storage

All cart data is automatically saved to browser's localStorage:

  • Key: cartData
  • Format: JSON array of cart items
  • Persistence: Survives page refresh/reload
  • Clearing: User can clear via "Clear Cart" button
